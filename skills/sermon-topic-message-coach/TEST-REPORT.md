### Test 1.3 — 메시지까지 정함: "고난을 주제로 '고난은 영광의 통로다' 메시지로"

- 행 2장 본문 확인: 1-47절 (행 2:1-13 성령강림, 2:14-41 베드로의 설교, 2:42-47 초대교회) — 본문 정확 ✓

**5단계 변형 시뮬레이션** (메시지 가정: "성령의 임재가 청년을 사도로 만든다"):

- 행 1:8 (성령이 임하시면 너희가 권능을 받고) — 1장 26절까지 ✓
- 행 4:31 (다 성령이 충만하여 담대히 하나님의 말씀을 전하니라) — 4장 37절까지 ✓

- 평행 사건: 엘리야가 엘리사에게 영의 갑절을 받게 함 (왕하 2:1-15), 70인 장로의 영을 받음 (민 11:24-30), 사울이 새 사람으로 변화 (삼상 10:6, 9-13)
