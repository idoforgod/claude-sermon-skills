# 할루시네이션 방지 체크리스트 — 5단계 성경 자료 추천 가드레일

본 체크리스트는 sermon-topic-message-coach의 **5단계(성경 구절 5개 + 성경 사건 5개 추천)**가 외부로 나가기 전 통과해야 할 4-Gate 검증이다. 단 한 항목이라도 실패하면 해당 항목을 수정하거나 "정확한 본문 직접 확인 권장"으로 대체한다.

5단계가 본 스킬에서 할루시네이션 위험이 가장 큰 영역이다 — 사용자가 정한 주제·메시지에 맞춰 성경 본문을 끌어다 추천하기 때문에, 본문이 정말 그 메시지를 가르치는지·인용 본문이 정확한지·사건의 근거 본문이 일치하는지 모두 검증되어야 한다.

---

## Gate 1: 성경 구절 인용의 텍스트 정확성

추천 5개 구절을 제시할 때:

- [ ] 책 이름·장·절 표기가 정확한가? (예: 엡 4:32, 골 3:13, 마 18:21-22)
- [ ] 인용한 한 문장이 개역개정 또는 표준 번역본과 실제로 일치하는가?
- [ ] 절 범위(예: 4:32 / 4:32-5:2)가 정확하며, 그 범위가 인용 의미를 자르지 않는가?
- [ ] 책 이름 약어가 표준인가? (창·출·레·민·신·수·삿·룻·삼상/하·왕상/하·대상/하·스·느·에·욥·시·잠·전·아·사·렘·애·겔·단·호·욜·암·옵·욘·미·나·합·습·학·슥·말 / 마·막·눅·요·행·롬·고전/후·갈·엡·빌·골·살전/후·딤전/후·딛·몬·히·약·벧전/후·요일/이/삼·유·계)

**금지**:
- "엡 4:35"처럼 존재하지 않는 절 번호 표기 (엡 4장은 32절까지)
- 표준 번역에 없는 어휘를 그 번역의 인용으로 표기
- 본문이 다른 책에 있는데 다른 책으로 표기 (예: 막 6장 사건을 마태로 표기)
- 한 절을 인용하면서 실은 다른 절 내용을 가져오기

**안전 규칙**:
- 한 문장 인용은 본인이 정확히 외우고 있는 본문에 한해. 정확하지 않으면 "(개역개정)" 표기를 붙이지 말고 "[본문 요지]"로 대체.
- 사용자가 받은 본문을 직접 펴서 확인할 수 있게 **장·절 표기는 반드시 정확**해야 한다.

## Gate 2: 성경 사건 추천의 근거 본문 일치

추천 5개 사건을 제시할 때:

- [ ] 사건 이름이 표준 명칭인가? (예: "요셉이 형들을 용서함", "다윗과 시므이", "탕자의 비유")
- [ ] 사건의 근거 본문(장 단위)이 정확한가?
- [ ] 사건이 그 본문에 실제로 기록되어 있는가? (다른 책으로 혼동 금지)
- [ ] 사건의 주인공·시대가 정확한가? (예: 사울 왕 시대인지 다윗 왕 시대인지)
- [ ] 본문에 없는 세부 묘사(인물 심리·미기록 대사·후일담)를 사건 설명에 추가하고 있지 않은가?

**검증된 사건 예시** (메시지별로 자주 추천되는 사건, 모두 본문 확인 완료):

### 용서·관계 회복 메시지
- 요셉이 형들을 용서함 (창 45:1-15, 50:15-21)
- 모세가 아론과 미리암을 위해 중보 (민 12:1-15)
- 다윗이 시므이를 용서함 (삼하 16:5-14, 19:16-23)
- 탕자의 비유 (눅 15:11-32)
- 예수님이 십자가에서 "아버지여 저들을 사하여 주옵소서" (눅 23:34)
- 스데반이 돌 던지는 자들을 위해 기도 (행 7:60)
- 베드로의 "일곱 번을 일흔 번까지라도" 가르침 (마 18:21-35, 무자비한 종 비유)

### 믿음·신뢰 메시지
- 아브라함이 이삭을 바침 (창 22:1-19)
- 출애굽 시 홍해 건너기 (출 14)
- 다윗과 골리앗 (삼상 17)
- 백부장의 믿음 (마 8:5-13, 눅 7:1-10)
- 가나안 여자의 믿음 (마 15:21-28, 막 7:24-30)
- 혈루증 여인 (마 9:20-22, 막 5:25-34, 눅 8:43-48)

### 회개·돌이킴 메시지
- 다윗이 나단 선지자 앞에서 회개 (삼하 12:1-15, 시 51편 표제)
- 요나의 니느웨 회개 (욘 3)
- 마리아 막달레나의 향유 (눅 7:36-50; 단, 마리아 막달레나라는 동일시는 학계 다수설이 *아님*)
- 베드로가 닭 울음 후 통곡 (마 26:75, 막 14:72, 눅 22:62)
- 삭개오 (눅 19:1-10)
- 바울의 다메섹 회심 (행 9:1-19, 22:6-16, 26:12-18)

### 은혜·자비 메시지
- 호세아가 고멜을 데려옴 (호 1-3장)
- 선한 사마리아인 (눅 10:25-37)
- 잃은 양·잃은 동전·잃은 아들 (눅 15)
- 간음하다 잡힌 여인 (요 7:53-8:11; 단, *Pericope Adulterae*는 사본학적으로 후대 첨가설이 다수설 — 강해는 신중)
- 십자가의 강도 (눅 23:39-43)

### 소망·종말 메시지
- 다니엘의 사자굴 (단 6)
- 사드락·메삭·아벳느고의 풀무불 (단 3)
- 요한계시록의 새 하늘 새 땅 (계 21-22)
- 나사로 살리심 (요 11)
- 부활 사건 (마 28, 막 16, 눅 24, 요 20-21)

### 감사·찬양 메시지
- 한나의 노래 (삼상 2:1-10)
- 다윗의 시편 — 시 23, 시 100, 시 103, 시 145
- 마리아의 송영 *Magnificat* (눅 1:46-55)
- 시므온의 송영 *Nunc Dimittis* (눅 2:29-32)
- 사가랴의 송영 *Benedictus* (눅 1:68-79)
- 한센병자 열 명 중 한 사마리아인 돌아옴 (눅 17:11-19)

### 헌신·순종 메시지
- 아브라함의 부르심 (창 12:1-9)
- 모세의 부르심 (출 3-4)
- 이사야의 부르심 (사 6)
- 마리아의 응답 "주의 여종이오니 말씀대로 내게 이루어지이다" (눅 1:38)
- 예수님의 겟세마네 "내 원대로 마옵시고 아버지의 원대로 하옵소서" (마 26:39, 막 14:36, 눅 22:42)
- 베드로의 그물 던짐 (눅 5:1-11)
- 부자 청년 (마 19:16-22, 막 10:17-22, 눅 18:18-23) — 반례 사례로 가능

### 사랑·이웃 메시지
- 룻과 나오미 (룻 1-4)
- 다윗과 요나단의 언약 (삼상 18-20)
- 선한 사마리아인 (눅 10:25-37)
- 발 씻기심 (요 13:1-17)
- 베드로 회복 "네가 나를 사랑하느냐" (요 21:15-19)

**금지**:
- 사건 이름은 잘 알려져 있으나 본문 위치를 임의로 표기 (예: 솔로몬의 재판 → "왕상 4장"이 아니라 정확히 *왕상 3:16-28*)
- 외경(*Apocrypha*)·랍비 문헌·교부 문헌 사건을 성경 사건으로 표기
- 본문에 없는 사건 (예: "모세의 어린 시절 학교 다닌 일") 추천

**안전 규칙**:
- 위 검증 목록에 없는 사건을 추천할 때는 사용자에게 정확한 본문 위치를 제시하고, 그 위치가 확실하지 않으면 일반 위치만 표기 ("창세기 50장 후반부에 형제 화해 본문" 식).

## Gate 3: 본문 → 메시지 연결의 타당성 (proof-texting 금지)

추천한 구절·사건이 사용자의 핵심 메시지를 *실제로 가르치는지* 검증:

- [ ] 본문의 1차 의미(*sensus literalis*)가 그 메시지를 직접 가르치는가?
- [ ] 비슷한 단어가 있을 뿐 맥락이 다른 본문을 끌어들이고 있지 않은가?
- [ ] 우의적·도덕적 적용에만 의존하지 않는가?
- [ ] 본문의 직접 맥락(앞뒤 절·책 전체 흐름)이 메시지와 일치하는가?

**proof-texting 회피 사례**:
- 메시지: "주님과 함께라면 어떤 어려움도 극복할 수 있다"
  - 빌 4:13 "내게 능력 주시는 자 안에서 내가 모든 것을 할 수 있느니라" → 
  - 1차 의미 점검: 바울이 *궁핍과 풍부 어느 처지에서도 자족하는 비결*을 말한 맥락 (빌 4:11-12 직전)
  - 추천 가능하되, "성공·승리"의 만능열쇠로 호도하지 않도록 메시지·맥락 명시

- 메시지: "하나님은 우리에게 좋은 계획을 가지고 계신다"
  - 렘 29:11 → 
  - 1차 의미 점검: 바벨론 포로 70년 백성을 향한 약속 (렘 29:10 직전, 29:14 회복 약속) — *개인 직업·진로 약속*이 아님
  - 추천 시 맥락 보존 필수

**금지**:
- 한 단어의 표면적 유사성만으로 본문 추천 (예: 메시지에 "사랑"이 있다고 "하나님은 사랑이심이라" 요일 4:8 + 다윗·요나단 "사랑" + 부부 "사랑"을 같은 결로 묶기)
- 본문 1차 의미와 다른 메시지를 본문이 가르친다고 진술
- 메시지에 본문을 끼워 맞추기

## Gate 4: 사건의 신학적 위치·해석 전통

추천한 사건이 정통 기독교 해석 전통과 일치하는지:

- [ ] 추천 사건의 표준 해석이 개혁주의 정통 또는 보편 교회 정통과 일치하는가?
- [ ] 외경·위경·랍비 전승·이단 해석을 본문 사건으로 끌어들이고 있지 않은가?
- [ ] 영지주의·신천지·통일교·여호와의증인 등 이단의 본문 재해석을 추천하고 있지 않은가?

**경계선 사례**:

- *Pericope Adulterae* (요 7:53-8:11, 간음하다 잡힌 여인)
  - 사본학적으로 후대 첨가 (다수설) — 가장 오래되고 권위 있는 사본(P66, P75, 시내·바티칸 사본)에 없음
  - 그러나 교회 전통에서 인정되어 강단에서 자주 사용
  - 추천 시 사용자에게 사본학적 위치를 *간단히* 알릴 수 있음. 강제는 아님.

- 막 16:9-20 (긴 결말)
  - 사본학적으로 후대 첨가 (다수설) — 시내·바티칸 사본 등에 없음
  - 뱀을 집고 독을 마셔도 해를 받지 않음 등은 특히 강단 추천 신중
  - "예수님 부활 후 등장" 일반 사건으로 추천 시 막 16:1-8까지로 한정 권장

- 눅 23:34a ("아버지여 저들을 사하여 주옵소서 자기들이 하는 것을 알지 못함이니이다")
  - 사본학적 이슈 있음 — 일부 초기 사본(P75, 시내 사본 1차 필사 등)에 빠져 있음. 그러나 다수의 사본 전통과 정통 강단에서 인정되어 사용. NA28에는 [이중 대괄호] 표기로 수록.
  - 추천 시 사용 가능하되, 사본학적 위치를 *간단히* 알릴 수 있음 (강제는 아님). 본문 자체의 메시지(원수 사랑·중보·용서)는 다른 본문(예: 마 5:44, 행 7:60 스데반)과 공통.

- 솔로몬과 시바 여왕 (왕상 10:1-13)
  - 본문 외 후대 전승(에티오피아 케브라 나가스트의 솔로몬·시바 자손 전승)을 본문 사건으로 끌어들이지 말 것

**금지**:
- 외경 사건(예: 토비트의 천사 동행, 유딧의 홀로페르네스 참수)을 정경 성경 사건으로 추천
- 영지주의 도마복음·마리아복음의 사건을 정경 사건으로 추천
- 이단 해석(예: 신천지의 "두 증인 = 이만희" 식 비유 해석)을 추천에 끌어들임

## Gate 5 (보조): 추천의 균형

5단계 SKILL.md 명시 선정 기준 재확인:

- [ ] 구절 5개 중 구약 1-2개, 신약 3-4개 균형이 잡혔는가? (메시지가 구약적 주제일 때는 조정 가능 — 명시 필요)
- [ ] 사건 5개가 다양한 인물·시대에서 선정됐는가? (한 책에 몰리지 않게)
- [ ] 친숙한 사건(다윗·바울·예수)과 덜 알려진 사건(아비가일·바나바·드보라 등) 섞임이 있는가?
- [ ] 모든 추천이 메시지와 *직접* 연결되는가? (무리한 적용 회피)

## 종합 의사결정

| 상황 | 처리 |
|------|------|
| 4-Gate + Gate 5 모두 통과 | 추천 그대로 사용 |
| Gate 1 실패 (장절 표기 불확실) | "정확한 위치 직접 확인 권장" 표기 또는 추천 교체 |
| Gate 2 실패 (사건과 본문 불일치) | 사건 교체 또는 본문 교체 |
| Gate 3 실패 (proof-texting) | 추천 교체. 메시지와 맥락이 일치하는 본문으로 |
| Gate 4 실패 (이단·외경 혼입) | 추천 전체 제거. 정통 사건만 |
| Gate 5 실패 (균형 깨짐) | 한두 개 교체로 조정 |

## 산출 직전 최종 자기 점검

1. 추천한 5개 구절 모두 책·장·절 표기가 정확한가?
2. 추천한 5개 사건 모두 근거 본문이 정확하고, 본문에 실제 기록된 사건인가?
3. 본문 인용 문장이 표준 번역과 일치하는가? (확신 없으면 인용 생략)
4. 모든 추천이 메시지의 1차 의미와 일치하는가? (proof-texting 없음)
5. 외경·위경·이단 해석이 끼어들지 않았는가?

5개 점검 모두 통과한 추천만 산출.
